# EXP2_像素值修改
##
对像素RGB进行操作

## Pixel
像素具有两个属性

- 位置
- 色彩

## RGB
颜色范围（R, G, B） `（0, 0, 0）- （255, 255, 255）`

## GDAL获取图像通道
`GetRasterBnad()`

- `GetRasterBnad(1)` 获取红色通道
- `GetRasterBnad(2)` 获取绿色通道
- `GetRasterBnad(3)` 获取蓝色通道
